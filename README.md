# Vrinda Store Data Analysis Project

# Project Overview
This project analyzes Vrinda Store sales data using Microsoft Excel.  
The goal is to identify sales trends, customer behavior, and key insights to improve business performance.

# Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- Dashboard

# Key Insights
- Women customers contribute ~65% of total sales
- Maharashtra, Karnataka, and Uttar Pradesh are top-performing states
- Adult age group (30–49 years) contributes the highest sales
- Amazon, Flipkart, and Myntra are major sales channels (~80%)
- Set categories show highest demand

# Business Recommendations
- Target women customers aged 30–49
- Focus marketing on top states
- Offer promotions on major online platforms
